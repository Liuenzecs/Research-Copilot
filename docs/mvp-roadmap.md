# MVP Roadmap

## Phase 1
- Docs + folder skeleton + backend/frontend/CLI skeleton.

## Phase 2
- SQLite schema + initialization.
- Provider abstraction.
- Search (arXiv + Semantic Scholar default).
- arXiv PDF download.
- Include reflections/task tables in schema early.

## Phase 3
- PDF parsing/chunking.
- Quick/deep summaries.
- Translation APIs (key-field + segment unit model).
- Library listing.

## Phase 4
- Reflection backend + templates + timeline + task linkage.
- Reflection UI (editor/templates/timeline/cards).
- Brainstorm APIs.
- Repo finder robustness (token optional, rate-limit handling, README fallback, PWC best-effort).
- Reproduction planner with manual-confirmation safety.

## Phase 5
- Memory service + ReflectionMemory linking.
- Chroma semantic retrieval.
- Profile updater.

## Phase 6
- Frontend integration hardening.
- Backend tests.
- Windows scripts.
- README polish.
